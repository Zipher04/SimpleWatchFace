A simpe watch face forked modified from the default Kickstart watchface came with pebble time.
The added inforinformation are:
1. ISO datetime.
2. Weekday
3. Step
4. Current weather data from OpenWeatherMap. Get free API key from [link](https://home.openweathermap.org/api_keys) and input it into Config page.
5. Outermost ring for battery percentage. Yellow when not charging; green when charging.
6. Inner ring for step percentage compared with daily average for each hour. (Original Kickstart watchface function)
